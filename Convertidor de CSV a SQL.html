¿Qué puede hacer esta herramienta?

  • Se pueden crear instrucciones INSERT, UPDATE, DELETE, MERGE y SELECT.
  • CREAR TABLA, DROP TABLE, CREAR VISTA, DROP VIEW son opcionales.
  • La clave utilizada en ACTUALIZAR, ELIMINAR y FUSIONAR se especifica configurando la columna de clave.
  • Ahora es fácil fusionar csv en una tabla de base de datos utilizando la nueva función Generate MERGE.
  • El valor de un campo puede recortarse, ponerse en mayúsculas o minúsculas. Puede agregar texto personalizado alrededor del valor del campo utilizando la función de plantilla. Por ejemplo, si está utilizando Oracle y desea convertir un campo en formato AAAAMMDD a FECHA, utilice TO_DATE ({f}, 'AAAAMMDD'). Del mismo modo, para Sql Server, use CAST ({f} AS DATETIME). El valor del campo se sustituirá por {f}.
  • La versión actual admite algunos resultados específicos de la base de datos y admite bases de datos ORACLE, MYSQL, SQL SERVER, PostreSQL y ANSI SQL. Existen ligeras variaciones en la forma en que las diferentes bases de datos manejan NULL, cadenas vacías, DATES, etc.

¿Cuáles son mis opciones?

  • Puede especificar qué campos incluir y especificar el nombre del campo.
  • Puede optar por citar los nombres de sus columnas según corresponda para su motor de base de datos.
  • Puede ordenar la salida por nombre de columna.
  • Puede utilizar NULL para campos vacíos.
